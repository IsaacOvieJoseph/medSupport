# HealthAid | Community Medical Fundraising

HealthAid is a robust, scalable web application designed to help people with health issues generate funds for their medical treatments. The platform emphasizes transparency, verification, and direct-to-hospital payments.

## Key Features

- **Verified Fundraisers**: Every post goes through a rigorous verification process where medical reports and hospital details are validated.
- **Real-time Progress**: Live cash buildup tracking for every campaign.
- **Dedicated Accounts**: Each fundraiser has a dedicated virtual account to ensure funds are used only for the specified medical treatment.
- **Hospital Partnerships**: Integration with a directory of trusted hospital partners to expedite the approval process.
- **Transparent Communication**: Community comments and requests on each post for direct engagement.
- **Automated Notifications**: Fundraiser owners are notified of approval, weekly milestones, and goal achievements.

## Tech Stack

- **Frontend**: React.js with Vite
- **Styling**: Vanilla CSS with Modern Design Tokens
- **Icons**: Lucide React
- **Routing**: React Router DOM

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Verification Process

Users can start a fundraiser in three steps:
1. Provide basic details and target goal.
2. Upload medical documents (Doctor's Report, Hospital Details, Verification Video).
3. Wait for admin verification (contacting the hospital). Partner hospitals provide faster approval.
